üöÄ Quick Start Example

The simplest way to get started with the Semantic Self-Aware Kit using the framework startup approach. 


# Basic framework initialization
from semantic_self_aware_kit import SemanticFramework

# Initialize the complete framework
framework = SemanticFramework()
await framework.startup()

# The framework is now ready with all components
print(f"Framework initialized with {len(framework.components)} components")
