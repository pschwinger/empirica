Semantic Translation & Movement

The visual representation must adhere to a strict semantic model. The core principle is that movement and color change are the primary languages of the interface.

Movement Protocol

An agent's movement is always purposeful. It represents a handoff, assignment, or audit. The path of an agent should be a clear, animated "train track" connecting its starting node to its destination node.
