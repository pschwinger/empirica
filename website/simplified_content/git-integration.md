## Workflow Examples

### Single AI Session
1. **PREFLIGHT:** Commit initial state.
2. **INVESTIGATE:** Branch for reasoning.
3. **CHECK:** Merge validation.
4. **ACT:** Commit code changes.
5. **POSTFLIGHT:** Measure learning delta.

### Multi-AI Coordination
- **AI-1:** Pushes complete epistemic state.
- **AI-2:** Fetches and rebases.
- **Result:** Seamless handoff without manual summaries.

### Calibration
**Sentinel** measures efficiency of reasoning branches and merges the best path.
