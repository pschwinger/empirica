### Development Setup

```bash
# Clone repository
git clone https://github.com/Nubaeon/empirica.git
cd empirica

# Install in development mode
pip install -e ".[dev]"

# Run tests
pytest

# Build docs
cd docs && make html
```

**Requirements:**
- Python 3.8+
- Git
- SQLite

**See:** [Development Guide](https://github.com/Nubaeon/empirica/blob/main/docs/DEVELOPMENT.md)

## Bug Reports

### How to Report a Bug

**Before reporting:**
1. Check [existing issues](https://github.com/Nubaeon/empirica/issues)
2. Try latest version: `pip install --upgrade empirica-sdk`
3. Check [troubleshooting guide](https://github.com/Nubaeon/empirica/blob/main/docs/production/21_TROUBLESHOOTING.md)

**What to include:**

```markdown
### Bug Description
[Clear description of the issue]

### Steps to Reproduce
1. Create session: `empirica session-create --ai-id test`
2. Run preflight: `empirica preflight --session-id <ID> --prompt "..."`
3. Error occurs: [paste error]

### Expected Behavior
[What should happen]

### Actual Behavior
[What actually happens]

### Environment
- OS: [e.g., Ubuntu 22.04, macOS 14.0, Windows 11]
- Python: [e.g., 3.11.5]
- Empirica: [e.g., 0.9.0]
- Installation: [pip, source]

### Logs
[Paste relevant logs from ~/.empirica/logs/]

### Additional Context
[Screenshots, session IDs, etc.]
```

**Open issue:** [GitHub Issues](https://github.com/Nubaeon/empirica/issues/new)
