# System Prompts

This directory contains categorized system prompts and configuration files for Empirica AI agents.

## Directory Structure

### `development/`
System prompts focused on development workflows and coding assistance.
- `SYSTEM_PROMPT_DEV_COMPACT.md` - Compact development agent prompt (with MCP parameter examples)
- `SYSTEM_PROMPT_DEVELOPMENT.md` - Full development-focused agent prompt
- `SYSTEM_PROMPT_COMPRESSED.md` - Compressed version for token efficiency

### `ai-agents/`
AI agent-specific prompts and configurations.
- `AI_EMPIRICA_REMINDERS.md` - Key reminders for AI agents
- `CLAUDE.md` - Claude-specific system prompt
- `COPILOT_CLAUDE.md` - Claude Copilot configuration
- `MINIMAX.md` - Minimax AI configuration
- `QWEN.md` - Qwen model configuration
- `ROVODEV.md` - RovoDev configuration
- `SYSTEM_PROMPT_ADDITION_SESSION_CASCADE.md` - Session cascade additions
- `SYSTEM_PROMPTS_FOR_AI_AGENTS.md` - General AI agent prompts
- `USER_PROMPTS_FOR_EMPIRICA.md` - User-facing prompt patterns

### `comprehensive/`
Complete reference guides and detailed documentation.
- `COMPLETE_MCP_TOOL_REFERENCE.md` - Full MCP tool reference
- `COMPRESSED_PROMPT_USAGE.md` - Compressed usage examples
- `GENERIC_EMPIRICA_SYSTEM_PROMPT.md` - Generic system prompt template

### `quick-reference/`
Quick reference materials and summaries.
- `README.md` - Directory overview
- `SYSTEM_PROMPT_QUICK_REFERENCE.md` - Quick reference guide

## Usage

Select the appropriate prompt based on your use case:
- **Development tasks** → `development/` folder
- **AI agent configuration** → `ai-agents/` folder  
- **Detailed reference** → `comprehensive/` folder
- **Quick lookup** → `quick-reference/` folder

## Migration

This directory replaced the previous `user-guides/` location. All content has been reorganized into logical categories for better discoverability and maintenance.
